##   常用命令

- 创建分支
  - `git branch 分支名>`
  - `git branch -v`
- 切换分支
  - `git checkout -b 分支名`  （创建并切换）
- 合并分支
  - `git merge 分支名 `
- 删除分支
  - `git branch -D 分支名`
- 查看ssh密钥
  - `cat ~/.ssh/id_rsa.pub`
